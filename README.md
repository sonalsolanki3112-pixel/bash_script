# Bash Scripting Projects 🚀

## About
These are my personal Bash scripting projects built 
while learning Linux and DevOps from scratch.
Every script was written and tested by me on Ubuntu Linux.

## Scripts

### Beginner Level
| Script | Description | Concepts Used |
|--------|-------------|---------------|
| hello.sh | First bash script | echo, comments |
| greet.sh | Personal greeting script | variables, input, arguments |

### Intermediate Level
| Script | Description | Concepts Used |
|--------|-------------|---------------|
| check_disk.sh | Disk usage monitor | conditions, df command |
| check_ram.sh | RAM usage monitor | functions, free command |
| loops.sh | Loop practice | for loop, while loop |
| log_manager.sh | Log file manager | file I/O, grep |

### Advanced Level
| Script | Description | Concepts Used |
|--------|-------------|---------------|
| system_health.sh | Complete system monitor | functions, loops, files, grep |

## Skills Demonstrated
- Variables and arguments
- If/elif/else conditions
- For and while loops
- Functions and error handling
- File reading and writing
- grep text searching
- Real system monitoring
- set -euo pipefail safety
